# Servomotor-Angle-Controller-using-Arduino-

An **Arduino project demonstrating precise servo motor angle control using the Servo library.**

This project shows how to control a servo motor by moving it through multiple angles using Arduino.

---

## 📌 Overview

This project demonstrates how to control the **angle of a servo motor** using an Arduino board and the built-in **Servo library**.

The Arduino sends position commands to the servo motor, which rotates to specific angles with short delays between each movement. This project is useful for beginners who want to learn how servo motors work with Arduino.

---

## ⚙️ Components Required

- Arduino Board (Uno, Nano, Mega, etc.)
- Servo Motor (e.g., SG90 / MG90S)
- Jumper Wires
- Breadboard (optional)
- USB Cable

---

## 🔌 Circuit Connections

| Servo Wire | Arduino Connection |
|------------|--------------------|
| VCC  | 5V |
| GND | GND |
| Signal | Digital Pin **9** |

---

## ▶️ How to Run the Project

1. Install the **Arduino IDE**.
2. Connect your **Arduino board** to your computer.
3. Connect the **servo motor** according to the wiring table.
4. Copy the code into a new Arduino sketch.
5. Select the correct **Board and Port** in the Arduino IDE.
6. Upload the code to the Arduino.
7. Observe the servo rotating through different angles.
